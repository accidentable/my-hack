# data/regulations — 규정 원문 (RAG · verify 근거) · JB 본업 중심

JB금융그룹 본업(대출·예금·외국인 금융)에 맞춰 규정 세트를 구성했다. 투자광고는 보조/확장용으로만 둔다.

## 구성
| 파일 | 규정ID | scope | 내용 |
|---|---|---|---|
| `금소법_제22조_광고일반준수사항.txt` | 금소법-제22조-광고일반준수사항 | 공통 | 필수 안내·혜택/불이익 균형 전달 |
| `금소법_제22조_대출성상품_광고.txt` | 금소법-제22조-대출성상품-광고 | **대출(JB 본업)** | 일 단위 금리 오인·부수비용/연체 고지 누락 등 |
| `금소법_제22조_예금성상품_광고.txt` | 금소법-제22조-예금성상품-광고 | **예금(JB 본업)** | 최고 우대금리만 강조·조건 누락 등 |
| `금소법_제19조_설명의무.txt` | 금소법-제19조-설명의무 | 공통/**외국인** | 외국어 콘텐츠의 중요사항 누락·왜곡 |
| `금소법_제21조_부당권유금지.txt` | 금소법-제21조-부당권유금지 | 공통 | 단정·과장·근거 없는 비교 |
| `협회_투자광고_의무표시사항.txt` | 협회-투자광고-의무표시사항 | 투자(확장) | 원금손실·과거실적 — 본업 아님, 확장용 |
| `reg_index.json` | — | — | 규정ID → 제목/출처/파일/scope (verify용) |

## JB 정합성 메모
- JB의 광고는 대부분 **대출·예금·카드**다(JB우리캐피탈 자동차금융, 전북/광주은행 여수신). 그래서 핵심은 대출성·예금성 광고 규정.
- **외국인 금융 1위**(전북은행)라는 JB 차별점은 `금소법-제19조-설명의무`로 연결 — 외국어 콘텐츠에서 위험·비용 고지가 누락·왜곡되는지가 JB의 진짜 리스크.
- 투자광고(협회 규정)는 JB자산운용 등으로 **확장할 때만** 쓰는 보조 규정.

## 데모 시나리오 권장(JB 맞춤)
- 입력 예: **베트남어 자동차/신용 대출 카드뉴스**
- 심어둘 위반 예:
  1. 대출이자를 "하루 ○○원"으로 표시 → `금소법-제22조-대출성상품-광고`
  2. 위험·연체·수수료 등 중요사항이 외국어로 누락 → `금소법-제19조-설명의무`
  3. "100% 승인", "무조건" 등 단정·과장 → `금소법-제21조-부당권유금지`

## 규칙엔진(rules.yaml) 연결 권장 매핑
- 일 단위 금리 오인 표시 → `금소법-제22조-대출성상품-광고`
- 최고금리만 강조·조건 누락(예금) → `금소법-제22조-예금성상품-광고`
- 외국어 중요사항 누락 → `금소법-제19조-설명의무`
- "무조건/100%/확정" 단정·과장 → `금소법-제21조-부당권유금지`
- 설명서·약관 권유 등 필수 안내 누락 → `금소법-제22조-광고일반준수사항`
